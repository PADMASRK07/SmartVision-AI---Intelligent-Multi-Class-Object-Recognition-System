🚀 SmartVision AI - Intelligent Multi-Class Object Recognition System
📌 Overview

SmartVision AI is a computer vision platform that performs multi-class image classification and object detection using deep learning. It combines CNN-based transfer learning models with YOLOv8 to accurately detect and classify multiple objects in real-world images.

🎯 Key Features
🔍 Multi-class image classification (25 classes)
📦 Object detection with YOLOv8 (bounding boxes + labels)
⚡ Fast and scalable inference
📊 Model comparison dashboard
🌐 Deployed using Streamlit & Hugging Face Spaces
🧠 Models Used
Classification (Transfer Learning)
VGG16
ResNet50
MobileNetV2
EfficientNetB0
Object Detection
YOLOv8 (Ultralytics)

📊 Dataset
COCO 2017 Dataset (25-class subset)
~2,500 images (100 per class)
Categories: Vehicles, Animals, People, Food, Indoor objects

⚙️ Tech Stack
Python, TensorFlow / PyTorch
OpenCV, NumPy, Pandas
Streamlit (UI)
Hugging Face (Dataset + Deployment)
YOLOv8 (Object Detection)

📈 Results
Classification Accuracy: ~80% – 93%
YOLOv8 mAP@0.5: ~85%+
Real-time detection with fast inference
